# gerrardhm.com

Personal portfolio website for Gerrard Hiren Michael. Built as a single-page static site, hosted on GitHub Pages.

## Local development

Open `index.html` in a browser.

## Deployment

Deployed automatically via GitHub Pages on push to `main`. Custom domain configured at `gerrardhm.com`.

## Updating photos

Add images to the relevant folder under `photos/` and update `index.html` with the new `<img>` tags.
